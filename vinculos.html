TEMA 3

VINCULOS

La sintaxis general de la etiqueta es muy simple: A HREF=”destino”>…/A>.
(Los puntos suspensivos representan el elemento enlazado.)
Para que el destino se abra en una ventana nueva, hay que indicar el siguiente
atributo: A HREF=”destino” target=”_blank”>…/A>

El destino de un hiperenlace puede ser de tres tipos:

Enlace externo: otro documento HTML situado en un ordenador remoto.
Enlace local: otro documento HTML situado en el mismo ordenador.
Ancla: otro lugar dentro del mismo documento HTML.
